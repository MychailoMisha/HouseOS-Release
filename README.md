# 🚀 HouseOS 3.6 Stable

HouseOS is an experimental operating system built from scratch using Rust and Assembly.

### 📜 Development History

* **1.x** — Kernel and graphics foundation
* **2.x** — Desktop, windows and applications
* **3.x** — File systems, browser, multimedia and networking

### ✨ Current Features

* Desktop Environment
* File Explorer
* Browser
* FAT32 & NTFS Support
* Image Viewer
* Video Player
* Music Player
* Text Editor
* Calculator
* Internet Support
* Light & Dark Themes
* Sound Driver (WIP)
